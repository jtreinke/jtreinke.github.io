Fall Frondescence

Fall foliage application



For my Interactive Design course, I built a prototype application that could be used for multiple devices such as web and mobile. Fall Frondescence is an interactive app that provides nation-wide fall foliage information. Basic services include an interactive map to display peak color times, a gallery review for fall colors at national parks, trails, and camping sites, in addition to tree facts. This project focused on the planning, interactive, and design process by creating concept maps, wireframes, and using prototyping tools. To learn more.

Software used: Adobe Illustrator, Photoshop, InDesign, and Invision
