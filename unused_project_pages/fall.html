Fall Frondescence

Prototype for Fall Foliage Application


For my Interactive Design course, I built a prototype application that could be used for multiple devices such as web and mobile. Fall Frondescence is an interactive app that provides nation-wide fall foliage information. Basic services include an interactive map to display peak color times and site locations, a gallery review for fall colors at national/state parks, trails, and camping sites, in addition to tree facts. This project focused on the planning, interactive, and design process by creating concept maps, wireframes, and using prototyping tools.

Software: Adobe Illustrator, Photoshop, InDesign, and Invision
